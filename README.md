# The Last-Minute Life Saver

## 🚀 Project Overview
The Last-Minute Life Saver is an AI-powered productivity companion that helps students, professionals, and entrepreneurs avoid missed deadlines. Unlike traditional tools that rely on passive reminders, this app proactively categorizes tasks, schedules them intelligently, and motivates users with gamification.

---

## 📌 Problem Statement
People frequently miss deadlines, assignments, meetings, and bill payments because existing productivity tools only send reminders without helping users take meaningful action.

---

## 💡 Solution
Our solution is an AI companion that:
- Categorizes tasks by urgency and importance.
- Suggests smart schedules based on deadlines and workload.
- Provides context-aware reminders.
- Tracks goals and habits with gamification features.

---

## 🔑 Key Features
- Intelligent task prioritization (Urgent / Important / Routine)
- AI-powered scheduling assistance
- Context-aware reminders
- Goal and habit tracking
- Gamification (streaks, points, badges)
- Calendar integration
- Voice-enabled task input (stretch feature)
- Autonomous task breakdown (stretch feature)
